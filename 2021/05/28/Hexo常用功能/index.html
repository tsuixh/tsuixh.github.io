Hexo常用功能

  • hexo new "My Post"创建新的博客文件
  • hexo init 初始化博客项目
  • hexo g 生成网页
  • hexo s 启动本地服务器
  • hexo d 部署到git中，根据配置

在MacOS上，全局安装了hexo之后，需要使用npx hexo来使用hexo。
